## TROUBLESHOOTING

### Build Failed: Flask not found
```
FIX: Make sure Procfile has correct path to requirements-prod.txt
    In Render: Settings → Build Command
    Should be: pip install -r requirements-prod.txt
```

### 500 Error on Login
```
FIX 1: Check SECRET_KEY is set
       Render → Environment → SECRET_KEY should exist

FIX 2: Verify app.py has password hashing code
       Search for: from werkzeug.security import generate_password_hash
```

### CORS Errors
```
FIX: Add to app.py:
     from flask_cors import CORS
     CORS(app)
     
Add to requirements-prod.txt:
     Flask-CORS==4.0.0
```

### Database Errors
```
FIX: SQLite is local file-based
     Data resets on redeploy
     Solution: Use Supabase PostgreSQL instead
              Set DATABASE_URL in Render
```

### API Returns 401 (Unauthorized)
```
EXPECTED: If not logged in
FIX 1: Make sure you're sending session cookie
FIX 2: Test login first, then use API endpoint
FIX 3: Check: curl -b "session=value" http://url/api/...
```
